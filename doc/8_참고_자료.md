# 참고 자료 및 개발 리소스 (References)

AdvanceKeep 프로젝트 개발에 사용된 주요 기술 스택의 공식 문서와 참고 사이트 목록입니다.

*   **Vite** (빌드 도구): [https://ko.vitejs.dev/](https://ko.vitejs.dev/)
*   **MDN Web Docs** (웹 표준 API 참고): [https://developer.mozilla.org/ko/](https://developer.mozilla.org/ko/)
