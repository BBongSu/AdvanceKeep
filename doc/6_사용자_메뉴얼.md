# 사용자 메뉴얼 (User Manual)

## 환영합니다!
**AdvanceKeep**은 당신의 아이디어를 쉽고 빠르게 기록할 수 있는 공간입니다.

## 1. 시작하기
## 1. 시작하기

### 회원가입 및 로그인
1.  웹 브라우저(Chrome, Safari 등)를 엽니다.
2.  **[https://advancekeep-1e051.web.app](https://advancekeep-1e051.web.app)** 주소로 접속합니다.
3.  다음 방법 중 하나로 로그인합니다:
    -   **이메일/비밀번호**: 회원가입 후 이메일과 비밀번호로 로그인
    -   **구글 계정**: "Google로 로그인" 버튼을 클릭하여 간편 로그인
4.  **로그인 상태 유지** 체크박스를 선택하면 브라우저를 닫아도 로그인 상태가 유지됩니다.


## 2. 기본 사용법

### 메모 작성하기
1.  화면 상단의 **"메모를 입력하세요..."** 영역을 클릭합니다.
2.  **제목**과 **내용**을 입력합니다.
3.  작성이 완료되면 **닫기** 버튼을 누르거나 입력창 바깥 빈 공간을 클릭하면 자동 저장됩니다.

### 이미지 추가하기
1.  메모 작성 중 또는 수정 중에 하단의 **이미지(🖼️) 아이콘**을 클릭합니다.
2.  컴퓨터에서 원하는 이미지 파일을 선택합니다.
3.  메모에 이미지가 첨부됩니다.

### 메모 배경색 변경하기
1.  메모 작성 중 또는 수정 중에 하단의 **물방울(💧) 아이콘**을 클릭합니다.
2.  나타나는 색상 팔레트에서 원하는 배경색을 선택합니다.
3.  선택한 색상이 즉시 메모에 적용됩니다.

### 라벨로 관리하기
1.  **라벨 생성**: 사이드바 하단의 **"라벨 수정"**을 클릭하여 새 라벨을 만들 수 있습니다.
2.  **라벨 추가**: 메모 작성/수정 시 하단의 **더보기(:)** 메뉴에서 **"라벨 추가"**를 선택하여 메모에 라벨을 붙일 수 있습니다.
3.  **라벨 필터링**: 사이드바에서 원하는 라벨 이름을 클릭하면 해당 라벨이 붙은 메모만 모아볼 수 있습니다.

### 메모 수정하기
1.  수정하고 싶은 메모 카드를 클릭합니다.
2.  내용을 수정합니다.
3.  **닫기**를 누르면 수정사항이 저장됩니다.

### 메모 삭제하기
1.  삭제할 메모 카드 하단의 **더보기(:)** 메뉴를 클릭합니다.
2.  **"메모 삭제"**를 선택합니다.
3.  메모가 **휴지통**으로 이동합니다. (완전히 삭제되지 않습니다)

## 3. 고급 기능

### 뷰 방식 변경 (리스트/그리드)
-   화면 우측 상단의 **뷰 전환 아이콘**을 클릭하면, 메모를 **바둑판식(그리드)** 또는 **목록형(리스트)**으로 변경하여 볼 수 있습니다.

### 검색하기
-   상단 검색창에 찾고 싶은 단어를 입력하면, 해당 단어가 포함된 메모만 실시간으로 보여줍니다.

### 사이드바 메뉴
-   **Notes**: 모든 활성 메모를 보여줍니다.
-   **Archive (보관함)**: 보관된 메모들을 확인합니다.
-   **Trash (휴지통)**: 삭제된 메모들을 확인하고 복구하거나 영구 삭제할 수 있습니다.
